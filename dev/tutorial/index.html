Tutorial

MyExample is a minimal working example using Documenter.jl and Literate.jl. The charts were prepared using CairoMakie.

The package has no functionality, it is solely for the purposes of testing the documentation generation workflow.

push!(LOAD_PATH, "../../src/")
using MyExample
