**MyExample** provides a minimal working example using [Documenter.jl](https://documenter.juliadocs.org/stable/) and [Literate.jl](https://fredrikekre.github.io/Literate.jl/v2/) following [https://documenter.juliadocs.org/stable/man/guide/](https://documenter.juliadocs.org/stable/man/guide/).

The plotting functionality comes from the CairoMakie tutorial [https://docs.makie.org/v0.21/tutorials/getting-started](https://docs.makie.org/v0.21/tutorials/getting-started)

See the [Tutorial](@ref tutorial) for how to use this package. Please note, however, that this package has no functionality and is solely for the purposes of testing the documentation generation workflow for a small package.
